# Contributing

## Feature Requests

Lilypad is a small community fork. You can file a feature request for a feature that you plan to implement and submit a pull request for, so that the feature can be reviewed and you will know ahead of time if the feature will be rejected.

## Bugs

Please search through the existing issues, open and closed, before filing a new bug.
Add the version of Lilypad, the version of the OS, and screenshots or videos as necessary.

## Coding Style

Please match the existing coding style as much as possible.

## License

By contributing to Lilypad you agree that your contributions will be licensed under its MIT license.

## Upstream

Lilypad is a fork of [Rectangle](https://github.com/rxhanson/Rectangle) by Ryan Hanson. If your change is general-purpose (not specific to Lilypad's grid/zone features), please consider contributing it upstream to Rectangle as well — the upstream project also offers free [Multitouch](https://multitouch.app) and [Rectangle Pro](https://rectangleapp.com/pro) licenses to significant contributors.
