# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately via GitHub's
[Security Advisories](https://github.com/bufothefrog/Lilypad/security/advisories/new)
("Report a vulnerability"). We'll acknowledge receipt and work with you on a fix
and coordinated disclosure.

## Scope

Lilypad runs locally and requires Accessibility (and, for chord mode, Input
Monitoring) permissions to move windows. Its only outbound network connection is
the Sparkle update check against the app's appcast feed. There is no telemetry
or analytics.

## Supported versions

The latest released version is supported. Older versions receive fixes on a
best-effort basis.
